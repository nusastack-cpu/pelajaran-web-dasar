# HTML Tables Mastery

Repositori ini fokus pada pembuatan tabel yang rapi dan terstruktur dalam HTML.

## Materi Utama:
- **`table`, `tr`, `th`, `td`**: Elemen dasar pembentuk tabel.
- **`thead`, `tbody`, `tfoot`**: Mengatur struktur semantik tabel.
- **`colspan`**: Menggabungkan sel secara horizontal (kolom).
- **`rowspan`**: Menggabungkan sel secara vertikal (baris).
- **Styling Dasar**: Memberikan warna, border, dan efek hover pada tabel.

## Cara Penggunaan:
Jalankan file `index.html` untuk melihat visualisasi tabel beserta kodenya.

## Penting:
Gunakan tabel hanya untuk data tabel (seperti laporan keuangan, jadwal, dll), **bukan** untuk membuat layout seluruh website (gunakan Flexbox/Grid untuk layout).
