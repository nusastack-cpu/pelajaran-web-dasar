# CSS Basics Mastery

Modul ini mempelajari dasar-dasar bagaimana CSS bekerja untuk mengatur tampilan elemen HTML.

## Apa yang Dipelajari?
1. **Selektor**: Cara memilih elemen menggunakan Tag, `.class`, dan `#id`.
2. **Sistem Warna**: Penggunaan nama warna, HEX, dan RGB.
3. **Tipografi**: Dasar-dasar pengaturan teks.
4. **The Box Model**: Konsep terpenting dalam CSS (Content, Padding, Border, Margin).

## Cara Menggunakan:
Buka `index.html`. Perhatikan bagaimana file tersebut terhubung ke `style.css` menggunakan tag `<link>`. Ini adalah praktik terbaik (Best Practice) dalam pengembangan web.

## Visualisasi Box Model:
Dalam contoh ini, saya menyertakan visualisasi berwarna:
- **Biru**: Konten (Isi).
- **Hijau**: Padding (Ruang di dalam).
- **Hitam Putus-putus**: Border (Garis tepi).
- **Merah**: Area Margin (Ruang di luar).
