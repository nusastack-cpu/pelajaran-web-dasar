3. Memahami Box Model

Setiap elemen web adalah sebuah kotak. Box model terdiri dari: Content, Padding, Border, dan Margin.

KOTAK KONTEN
  • Content: Isi dari elemen (teks/gambar).
  • Padding: Jarak antara konten dan border (di dalam).
  • Border: Garis tepi elemen.
  • Margin: Jarak antara elemen dengan elemen lain (di luar).
