# CSS Animations & Transitions

Modul ini mempelajari cara membuat elemen web berinteraksi dengan pengguna melalui gerakan yang halus.

## Konsep yang Dipelajari:
1. **`transition`**: Mengatur perubahan properti (seperti warna atau ukuran) agar tidak terjadi secara instan, melainkan dalam durasi waktu tertentu.
2. **`transform`**: Mengubah bentuk fisik elemen tanpa mengganggu tata letak elemen di sekitarnya (seperti `rotate`, `scale`, dan `translate`).
3. **`@keyframes`**: Cara membuat animasi kustom yang memiliki banyak langkah (state).
4. **`animation`**: Properti untuk menjalankan animasi yang telah didefinisikan di `@keyframes`.

## Contoh Praktis:
- **Button Hover**: Menggunakan transisi untuk efek bayangan dan pergeseran halus saat kursor diarahkan.
- **Loading Spinner**: Menggunakan animasi berulang (`infinite`) untuk membuat indikator proses yang berputar.

## Cara Menggunakan:
Buka `index.html` dan arahkan kursor Anda ke berbagai elemen untuk melihat efek transisinya.
