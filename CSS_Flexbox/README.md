# CSS Flexbox Mastery

Flexbox adalah sistem tata letak satu dimensi yang sangat kuat untuk mengatur elemen dalam baris atau kolom.

## Properti Utama (Parent):
- **`display: flex`**: Mengaktifkan mode flexbox pada kontainer.
- **`flex-direction`**: Menentukan arah item (`row` atau `column`).
- **`justify-content`**: Mengatur posisi item di sumbu utama (horizontal jika row).
- **`align-items`**: Mengatur posisi item di sumbu silang (vertikal jika row).
- **`gap`**: Memberikan jarak antar item tanpa perlu menggunakan margin manual.

## Kegunaan Utama:
1. Membuat Navbar (Menu navigasi).
2. Menaruh elemen tepat di tengah layar (Centering).
3. Membuat grid sederhana yang fleksibel.

## Cara Menggunakan:
Buka `index.html` dan perhatikan bagaimana elemen kotak-kotak tersebut diatur posisinya hanya dengan beberapa baris CSS di `style.css`.
