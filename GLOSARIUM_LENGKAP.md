# Glosarium Lengkap: Fungsi & Tag Web Development

Dokumen ini berisi daftar "catatan" untuk semua elemen, properti, dan fungsi yang digunakan dalam seluruh kurikulum ini.

---

## 🌎 HTML (Struktur)
| Tag | Fungsi |
|-----|--------|
| `<html>` | Akar dari seluruh dokumen HTML. |
| `<head>` | Tempat metadata (info rahasia browser, judul, link CSS). |
| `<body>` | Isi utama website yang dilihat pengguna. |
| `<ul> / <ol>` | Membuat daftar (Unordered / Ordered). |
| `<li>` | Item di dalam sebuah daftar. |
| `<table>` | Membuat kontainer tabel data. |
| `<tr> / <th> / <td>` | Baris / Judul Kolom / Sel Data dalam tabel. |
| `<form>` | Kontainer untuk formulir input pengguna. |
| `<input>` | Elemen untuk menerima teks, email, password, dll. |
| `<header>` | Bagian kepala (semantik). |
| `<nav>` | Bagian navigasi (semantik). |
| `<main>` | Konten utama yang unik (semantik). |
| `<section>` | Bagian/bab dalam satu halaman (semantik). |
| `<article>` | Konten mandiri seperti berita (semantik). |
| `<footer>` | Bagian kaki website (semantik). |

---

## 🎨 CSS (Tampilan)
| Properti | Fungsi |
|----------|--------|
| `color` | Mengubah warna teks. |
| `background-color`| Mengubah warna latar belakang. |
| `padding` | Ruang di **dalam** elemen (sebelum border). |
| `margin` | Ruang di **luar** elemen (jarak antar elemen). |
| `border` | Garis tepi elemen. |
| `display: flex` | Mengaktifkan tata letak fleksibel (1 dimensi). |
| `display: grid` | Mengaktifkan tata letak kotak-kotak (2 dimensi). |
| `justify-content` | Perataan horizontal di Flex/Grid. |
| `align-items` | Perataan vertikal di Flex/Grid. |
| `transition` | Membuat perubahan properti terasa halus/lambat. |
| `transform` | Memutar (`rotate`), memperbesar (`scale`), menggeser (`translate`). |
| `@keyframes` | Aturan untuk membuat langkah-animasi. |

---

## ⚡ JavaScript (Logika)
| Fungsi / Keyword | Fungsi |
|------------------|--------|
| `let / const` | Deklarasi variabel (bisa diubah / tetap). |
| `function` | Kumpulan perintah yang dibungkus agar bisa dipanggil ulang. |
| `console.log()` | Mencetak pesan ke terminal browser (untuk debug). |
| `document.getElementById()`| Memilih elemen HTML berdasarkan ID-nya. |
| `addEventListener()`| Menunggu aksi pengguna (seperti klik atau ketik). |
| `fetch()` | Mengambil data dari internet (API). |
| `async / await` | Menjalankan proses berat tanpa membekukan website. |
| `localStorage.setItem()`| Menyimpan data secara permanen di browser. |
| `localStorage.getItem()`| Mengambil data yang disimpan di browser. |
