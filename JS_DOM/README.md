# JavaScript DOM Manipulation Mastery

DOM (Document Object Model) adalah representasi objek dari halaman HTML Anda yang memungkinkan JavaScript untuk mengakses, mengubah, menambah, atau menghapus elemen dan gaya secara dinamis.

## Apa yang Dipelajari?
1. **Seleksi Elemen**: Cara memberitahu JS elemen mana yang ingin kita kendalikan menggunakan `getElementById`.
2. **Event Listeners**: Cara membuat website bereaksi terhadap aksi pengguna (seperti klik tombol atau mengetik di keyboard).
3. **Manipulasi Konten**: Merubah isi teks di dalam elemen menggunakan atribut `innerText`.
4. **Manipulasi Style**: Merubah CSS secara langsung melalui properti `.style` di JavaScript.

## Contoh di Modul Ini:
- **Ganti Judul**: Mendemonstrasikan perubahan teks sederhana.
- **Warna Acak & Toggle Bulat**: Mendemonstrasikan cara memanipulasi gaya CSS (background-color dan border-radius).
- **Interaksi Input**: Mendemonstrasikan cara mengambil nilai dari input teks secara real-time.

## Cara Menggunakan:
Buka `index.html` dan berinteraksilah dengan elemen-elemen di dalamnya untuk melihat bagaimana JavaScript merubah halaman secara instan tanpa perlu memuat ulang (refresh) halaman.
