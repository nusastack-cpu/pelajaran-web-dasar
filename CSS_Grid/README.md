# CSS Grid Mastery

CSS Grid adalah sistem tata letak paling canggih di CSS yang memungkinkan pembuatan layout dua dimensi (baris dan kolom secara bersamaan).

## Perbedaan dengan Flexbox:
- **Flexbox**: Satu dimensi (Hanya baris ATAU hanya kolom).
- **Grid**: Dua dimensi (Baris DAN kolom sekaligus).

## Properti yang Dipelajari:
- **`display: grid`**: Mengaktifkan mode grid.
- **`grid-template-columns`**: Menentukan jumlah dan ukuran kolom.
- **`grid-template-areas`**: Memberi nama pada area grid untuk mempermudah peletakan elemen layout besar.
- **`fr` (Fractional Unit)**: Unit baru yang fleksibel untuk membagi ruang yang tersedia.
- **`repeat()`**: Fungsi untuk mempermudah penulisan kolom yang berulang.

## Responsivitas:
Dalam contoh `style.css`, saya telah menyertakan **Media Query**. Jika layar diperkecil, tata letak grid akan otomatis berubah dari 2 kolom menjadi 1 kolom.

## Cara Menggunakan:
Buka `index.html` dan coba ubah ukuran jendela browser Anda untuk melihat bagaimana layout tersebut berubah secara responsif.
