# HTML Semantic Mastery

Modul ini mempelajari pentingnya menggunakan tag yang bermakna (Semantik) daripada hanya menggunakan `<div>` secara berlebihan.

## Tag yang Dipelajari:
- **`<header>`**: Bagian kepala situs/artikel.
- **`<nav>`**: Kumpulan navigasi link.
- **`<main>`**: Isi utama dari halaman (hanya boleh ada satu per halaman).
- **`<section>`**: Mengelompokkan konten berdasarkan tema/topik.
- **`<article>`**: Konten mandiri yang bisa didistribusikan ulang (seperti berita atau post blog).
- **`<aside>`**: Konten pendukung yang berkaitan secara tidak langsung (seperti sidebar).
- **`<footer>`**: Bagian kaki situs (kontak, copyright, dll).

## Mengapa Menggunakan Semantic HTML?
1. **SEO (Search Engine Optimization)**: Membantu Google memahami website Anda.
2. **Aksesibilitas**: Membantu pembaca layar (screen reader) bagi penyandang disabilitas.
3. **Keterbacaan Kode**: Memudahkan developer lain membaca struktur web Anda.

## Cara Menggunakan:
Buka `index.html` dan perhatikan bagaimana struktur visual yang ditampilkan mencerminkan struktur kode yang benar.
