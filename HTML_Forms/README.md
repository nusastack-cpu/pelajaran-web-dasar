# HTML Forms Mastery

Modul ini mempelajari cara membuat formulir interaktif di website.

## Elemen yang Dipelajari:
- **`<form>`**: Kontainer utama untuk formulir.
- **`<label>`**: Memberikan label pada input (sangat penting untuk aksesibilitas).
- **`<input>`**: Berbagai macam tipe seperti `text`, `email`, `password`, `radio`, dan `checkbox`.
- **`<select>` & `<option>`**: Membuat menu dropdown.
- **`<textarea>`**: Input teks panjang (paragraf).
- **`<fieldset>` & `<legend>`**: Mengelompokkan input yang berkaitan secara visual.
- **`required`**: Atribut dasar untuk validasi agar input tidak boleh kosong.

## Cara Penggunaan:
Buka `index.html` dan cobalah mengisi formulir tersebut. Lihat bagaimana atribut `name` dan `id` bekerja sama dengan tag `<label>`.

## Penting:
Gunakan atribut `name` pada setiap input agar data dapat dikirimkan dan diproses oleh server di kemudian hari.
