# Bootstrap Mastery Guide

Bootstrap adalah kerangka kerja (framework) CSS paling populer di dunia untuk membangun website yang responsif dan berorientasi mobile (mobile-first) dengan cepat.

## Konsep Utama:
1. **CDN (Content Delivery Network)**: Cara tercepat menggunakan Bootstrap tanpa perlu mendownload filenya. Cukup hubungkan link CSS dan JS di dalam tag `<head>` dan sebelum penutup `</body>`.
2. **Grid System**: Menggunakan sistem 12 kolom (`row` dan `col`) untuk mengatur tata letak yang otomatis menyesuaikan ukuran layar.
3. **Utility Classes**: Kelas-kelas siap pakai seperti `bg-primary`, `shadow`, `p-4` (padding), `my-5` (margin vertikal), dll.
4. **Components**: Elemen UI yang sudah jadi seperti Navbar, Card, Button, dan Modal yang tinggal kita gunakan kodenya.

## Kenapa Menggunakan Bootstrap?
- **Hemat Waktu**: Tidak perlu menulis ribuan baris CSS dari nol.
- **Konsisten**: Tampilan website akan memiliki standar desain yang profesional.
- **Responsif Otomatis**: Website Anda akan langsung terlihat bagus di HP tanpa tambahan kode CSS manual yang rumit.

## Cara Menggunakan:
Buka `index.html`. Perhatikan bagaimana kelas-kelas Bootstrap (seperti `navbar`, `container`, `row`, `col-md-4`) bekerja untuk membentuk tata letak yang indah secara instan.
