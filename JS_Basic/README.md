# JavaScript Basics Mastery

Modul ini mempelajari dasar-dasar logika pemrograman menggunakan JavaScript.

## Konsep yang Dipelajari:
1. **Variabel (`let`, `const`)**: Tempat menyimpan data. `let` untuk data yang berubah, `const` untuk data tetap.
2. **Tipe Data**: String (teks), Number (angka), Boolean (true/false), dan Array (kumpulan data).
3. **Operator**: Penjumlahan, pengurangan, dan manipulasi data.
4. **Function**: Kumpulan instruksi yang dapat dipanggil kapan saja.
5. **Output**: Cara menampilkan data ke konsol browser (`console.log`) dan ke halaman web (`innerHTML`).

## Cara Menggunakan:
1. Buka `index.html`.
2. Anda akan melihat hasil eksekusi skrip di dalam kotak hitam.
3. Tekan **F12** (atau Klik Kanan > Inspect > klik tab **Console**) pada browser Anda untuk melihat output tambahan dari `console.log`.

## Penting:
Gunakan `const` sebanyak mungkin. Gunakan `let` hanya jika Anda yakin nilai variabel tersebut akan berubah nanti.
