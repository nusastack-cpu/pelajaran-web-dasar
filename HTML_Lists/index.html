Mastery HTML: Lists (Daftar)

Selamat datang di modul pembelajaran pertama! Di sini kita akan mempelajari semua jenis daftar di HTML secara mendalam.

PENTING: Daftar digunakan untuk mengelompokkan item yang memiliki hubungan satu sama lain.

1. Unordered List (<ul>)

Digunakan untuk daftar yang urutannya tidak penting. Biasanya menggunakan simbol bullet.

  • Nasi Goreng
  • Sate Ayam
  • Gado-gado
  • Rendang

2. Ordered List (<ol>)

Digunakan untuk daftar yang memiliki urutan khusus (langkah-langkah, ranking, dll).

  1. Siapkan bahan makanan
  2. Cuci bahan hingga bersih
  3. Masak dengan api sedang
  4. Sajikan selagi hangat

Kita juga bisa mengubah tipe angkanya (A, i, I, dll) menggunakan atribut type.

  1. Opsi Pertama
  2. Opsi Kedua

3. Description List (<dl>)

Digunakan untuk istilah dan definisinya (seperti kamus).

HTML
HyperText Markup Language - Bahasa standar untuk membuat halaman web.
CSS
Cascading Style Sheets - Digunakan untuk mengatur tampilan visual web.

4. Nested List (Daftar di Dalam Daftar)

Sangat berguna untuk membuat struktur menu atau kategori.

  • Minuman Dingin
    • Es Teh
    • Es Jeruk
    • Jus Alpukat
  • Minuman Panas
    • Kopi Hitam
    • Teh Manis Panas
