# HTML Lists Mastery

Repositori ini berisi materi pembelajaran lengkap mengenai penggunaan List (Daftar) dalam HTML.

## Apa yang akan dipelajari?
1. **Unordered List (`<ul>`)**: Daftar poin atau bullet.
2. **Ordered List (`<ol>`)**: Daftar berurutan dengan angka atau huruf.
3. **Description List (`<dl>`)**: Daftar istilah dan deskripsinya.
4. **Nested Lists**: Cara memasukkan daftar ke dalam daftar lainnya.

## Cara Menggunakan
Cukup buka file `index.html` di browser pilihan Anda untuk melihat contoh kode dan penjelasannya secara langsung.

## Catatan Responsivitas
Contoh ini sudah menggunakan CSS dasar agar tampilan tetap rapi di berbagai ukuran layar.
